
## Funcionalidades
- Carrossel de rotas
- Mapa interativo
- Categorias de lugares
- Adicionar lugares
- Adicionar rotas

## Pré-requisitos

- Flutter SDK (versão 3.16 ou superior)
- Dart SDK (versão 3.0 ou superior)
- VS Code
## Instalação

Clone o repositório

```
    git clone https://github.com/jeansprogrammer/boranemobile.git 
    cd boranemobile
```

## Instale as dependências

```
    flutter pub get
```

Execute o aplicativo
```
    flutter run
```
